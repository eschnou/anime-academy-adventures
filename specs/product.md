# Product Overview — Kaizen Academy

## Purpose
Kaizen Academy is an educational web application that makes learning STEM subjects fun and engaging for children through anime and ninja-themed gamification. It transforms traditional exercises into "missions" that kids complete to earn XP, level up, and progress through ninja ranks.

## Target Users
- **Primary:** Children ages 6–12+
- **Secondary:** Parents and educators looking for engaging STEM learning tools

## Core Value Proposition
Combines the appeal of anime/ninja culture with structured STEM education, turning math, physics, science, and biology lessons into an adventure. The gamified progression system (XP, ranks, missions) motivates continued learning through intrinsic rewards.

## Key Capabilities
- **Age-adaptive content:** An age gate filters topics and difficulty to match the learner's level
- **Multi-subject coverage:** Math, Physics, Science, and Biology topics organized by category
- **Gamified exercises:** Multiple-choice, fill-in-the-blank, and true/false exercises framed as ninja missions
- **Progress tracking:** XP accumulation, ninja rank progression, and per-topic completion tracking
- **Info scrolls:** Educational content presented as collectible "scrolls" within each topic
- **Anime-styled UI:** Custom theming with display fonts, animations, and a vibrant color palette

## Technical Context
- Single-page React application (no backend; all data is mock/in-memory)
- Built with React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui, Framer Motion, and React Router
- Designed for future backend integration via a structured mock API layer
